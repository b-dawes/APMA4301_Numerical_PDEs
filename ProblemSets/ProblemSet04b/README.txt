This folder contains scripts for problem set 4b. solvers.shml contains the simulation harness which loops over poisson_mms.tfml. Convergence_Ncells_{n}.pdf contains a convergence plot of all the solvers with Ncells=n. Truncation_Error.pdf shows the truncation error for each solver as a function of the number of cells. The LATEX writeup is in the subdirectory LATEX.
