This directory contains programs used for problem set 3. The latex source, pdf, and images can be found in the LATEX subfolder.

poisson2d_mms.py solves a poisson problem on the unit square with dirichlet boundary conditions and equal x and y meshes.

poisson2d_rectangle.py generalizes this problem to an arbitrary rectangle.

poisson2d_nonuniformMesh.py further generalizes this for different x and y meshes.

poisson2d_neumann.py solves this problem with neumann boundary conditions on the right side of the rectangle.
